# Dastuurka-26 — Nuxt-like (Astro + Tailwind + Sanity)

## Run website
```bash
npm install
npm run dev
```

## Locales
- English: `/en`
- Somali: `/so`

## Sanity (optional)
Copy `.env.example` → `.env` and set `PUBLIC_SANITY_PROJECT_ID` and `PUBLIC_SANITY_DATASET`.
Run studio:
```bash
npm run sanity:dev
```

## Somali Emblem
Replace `public/somalia-emblem.svg` with the official Somali emblem SVG.
